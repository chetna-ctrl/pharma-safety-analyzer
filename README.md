# Pharma-Safety Analyzer

Predictive AI dashboard for chemical toxicity and pharma risk analysis.

## Features
- ML-driven safety prediction (Random Forest & XGBoost).
- Advanced Structural Alert detection (Expert Hybrid Layer).
- Real-time property analysis using RDKit.

## Deployment
This app is designed to run on Streamlit.
```bash
pip install -r requirements.txt
streamlit run app.py
```
